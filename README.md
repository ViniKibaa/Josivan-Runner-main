# 🎮 Josivan Runner

Um jogo de plataforma 2D desenvolvido na **Unity** utilizando **C#**, criado como projeto acadêmico com o objetivo de aplicar conceitos de programação, física, colisões e desenvolvimento de jogos.

---

## 📖 Sobre o Projeto

O **Josivan Runner** é um jogo de plataforma 2D onde o jogador deve superar obstáculos e explorar o cenário utilizando mecânicas clássicas de movimentação.

Durante o desenvolvimento foram aplicados conceitos fundamentais de desenvolvimento de jogos, incluindo gerenciamento de cenas, física 2D, colisões, animações e programação orientada a objetos.

---

## ▶️ Como Executar

1. Clone o repositório

```bash
git clone https://github.com/ViniKibaa/Josivan-Runner-main.git
```

2. Abra o projeto na **Unity Hub**.

3. Utilize uma versão compatível da Unity (a mesma utilizada no projeto).

4. Abra a cena principal.

5. Clique em **Play**.
